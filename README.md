
# sirclo-test
[Berat APP] CRUD App using `Golang`, `Echo Framework`, and `Gorm`


## How To Run
[Berat APP]
Already installed inside `Bin` directory of `Berat APP` directory
- Run Command
```
./start.sh
```
*Note : if error is occur, try to put sirclo-test inside `$GOPATH/src directory`, then
```
./install.sh
```
